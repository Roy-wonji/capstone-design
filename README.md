## 1.개발진  

정보통신공학과 캡스톤디자인  류황교수님 ROOMS 
- IOS : 서원지(팀장)
- 디자인: 윤성준 , 곽동엽
- 웹 : 이상준  , 김우열 , 정지원

  ### Commit Convention

  ```
  <Type>: <Message>
  ```

  feat: 기능추가  
  feat: 기능  
  fix: 버그 수정  
  refactor: 리팩토링  
  style: 스타일(코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없음)  
  docs: 문서 (문서 추가, 수정, 삭제)  
  test: 테스트(테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없음)  
  chore: 기타 변경사항 (빌드 스크립트 수정 등
